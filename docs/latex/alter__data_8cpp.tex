\doxysubsection{Detailed Description}
This program modifies genotype data by introducing missing values based on parameters specified in an input file {\ttfamily parameters.\+txt}. 

The modified data is saved in a new CSV file.

Key Features\+:
\begin{DoxyItemize}
\item Reading a percentage of missing data to introduce from a parameters file.
\item Reading genotype data from a CSV file.
\item Introducing missing data into the genotype matrix.
\item Writing the altered genotype data to a new CSV file.
\end{DoxyItemize}

@output A new CSV file {\ttfamily \texorpdfstring{$<$}{<}genotypes\texorpdfstring{$>$}{>}\+\_\+altered.\+csv} containing the modified genotype data.

@preconditions
\begin{DoxyItemize}
\item {\ttfamily parameters.\+txt} must specify the percentage of missing data (as a float).
\item {\ttfamily genotypes.\+csv} must contain a valid genotype matrix.
\item Ensure proper file paths and directory permissions.
\end{DoxyItemize}
