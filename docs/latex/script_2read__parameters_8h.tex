\doxysubsection{Detailed Description}
Defines the function {\ttfamily read\+\_\+parameters} to read configuration parameters from a specified text file. 

This header file defines the function {\ttfamily read\+\_\+parameters}, which reads configuration parameters from a specified text file. The function extracts the values for the number of individuals ({\ttfamily n\+\_\+ind}), the number of loci ({\ttfamily n\+\_\+loci}), and the number of distinct haplotypes ({\ttfamily n\+\_\+distinct\+\_\+haplo}) from the file and assigns them to the respective variables.

\item It reads each line of the file and extracts the values of {\ttfamily n\+\_\+ind}, {\ttfamily n\+\_\+loci}, and {\ttfamily n\+\_\+distinct\+\_\+haplo}.
