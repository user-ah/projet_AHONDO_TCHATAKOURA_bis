This header file declares the function {\ttfamily calculate\+Hamming\+Distance}, which computes the Hamming distance between two haplotypes represented as vectors of integers. The Hamming distance is the number of positions where the corresponding elements of the two vectors differ.

The function ensures that both input haplotypes have the same length before performing the calculation. If they have different lengths, an exception is thrown.\hypertarget{write__genotypes__knn_8h_Features}{}\doxysubsection{\texorpdfstring{Key Features}{Key Features}}\label{write__genotypes__knn_8h_Features}
