\doxysubsection{Detailed Description}
Defines the function {\ttfamily save\+\_\+csv} for saving a 2D vector of integers into a CSV file. 

This header file contains the declaration of the {\ttfamily save\+\_\+csv} function, which writes a 2D vector of integers to a CSV file. The function ensures that values are separated by commas and rows are separated by newline characters.

\begin{DoxyAuthor}{Authors}
Ahamed TCHATAKOURA \& Mawu√©na AHONDO
\end{DoxyAuthor}
The {\ttfamily save\+\_\+csv} function performs the following operations\+:
\begin{DoxyItemize}
\item Opens the specified file for writing.
\item Iterates through each row of the 2D vector and writes the values to the file, adding commas between the values in a row.
\item Adds a newline character after writing each row to separate rows.
\item Closes the file after writing all the data.
\end{DoxyItemize}
