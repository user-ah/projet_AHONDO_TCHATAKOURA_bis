Imputes missing values in a genotype vector using a KNN-\/based approach. 

This function identifies the {\ttfamily k} nearest neighbors based on the Hamming distance between the {\ttfamily target} vector and all genotype vectors in the dataset. Missing values ({\ttfamily -\/1}) in the {\ttfamily target} vector are replaced using a majority vote from the corresponding positions in the nearest neighbors.
