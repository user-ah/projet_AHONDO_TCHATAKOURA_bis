This header file declares the function {\ttfamily calculate\+Average\+Hamming\+Distance}, which computes the Hamming distance for each pair of inferred haplotypes compared to a reference set (ground truth). The function returns the average normalized Hamming distance across all individuals.\hypertarget{write__genotypes__knn_8h_Features}{}\doxysubsection{\texorpdfstring{Key Features}{Key Features}}\label{write__genotypes__knn_8h_Features}

\begin{DoxyItemize}
\item Computes Hamming distances between haplotype pairs.
\item Normalizes distances by the number of loci.
\item Supports direct comparison of inferred haplotypes against a ground truth reference.
\end{DoxyItemize}\hypertarget{write__genotypes__knn_8h_Preconditions}{}\doxysubsection{\texorpdfstring{Preconditions}{Preconditions}}\label{write__genotypes__knn_8h_Preconditions}
