Main function implementing the Expectation-\/\+Maximization algorithm for haplotype inference. 

This function orchestrates the entire haplotype inference process, including\+:
\begin{DoxyItemize}
\item Reading input data (genotypes and haplotypes).
\item Initializing haplotype frequencies and computing genotype probabilities.
\item Performing EM iterations until convergence is reached.
\item Reordering haplotypes based on genotype mapping.
\item Computing the average Hamming distance for validation.
\item Writing results to output files.
\end{DoxyItemize}
