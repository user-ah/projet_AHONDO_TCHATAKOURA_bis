\doxysubsection{Detailed Description}
This header file defines the function {\ttfamily generate\+\_\+haplotypes}, which generates random haplotypes for a given population of individuals. 

\begin{DoxyAuthor}{Authors}
Ahamed TCHATAKOURA \& Mawu√©na AHONDO 
\end{DoxyAuthor}
The function creates a set of distinct haplotypes, then pairs them randomly to generate haplotypes for each individual.


\begin{DoxyItemize}
\item First, the function generates {\ttfamily n\+\_\+distinct\+\_\+haplo} distinct haplotypes, each having {\ttfamily n\+\_\+loci} loci, with each locus being randomly assigned a value of 0 or 1.
\item Then, it randomly selects pairs of haplotypes from the distinct haplotypes and assigns them to individuals.
\end{DoxyItemize}


\begin{DoxyParams}{Parameters}
{\em n\+\_\+ind} & The number of individuals (i.\+e., the number of haplotype pairs to generate). \\
\hline
{\em n\+\_\+loci} & The number of loci (genetic markers) in each haplotype. \\
\hline
{\em n\+\_\+distinct\+\_\+haplo} & The number of distinct haplotypes to generate.\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
A 2D vector representing the haplotypes for all individuals. Each individual has two associated haplotypes. 
\end{DoxyReturn}
