This header file provides functions to verify if two haplotypes can explain a given genotype based on Mendelian inheritance rules. The main function, {\ttfamily explain\+Genotype}, ensures that each allele in the genotype can be derived from the corresponding alleles in the haplotype pair.

A helper function, {\ttfamily is\+Valid}, is used to check individual allele compatibility, ensuring that the inheritance constraints are met.\hypertarget{write__genotypes__knn_8h_Features}{}\doxysubsection{\texorpdfstring{Key Features}{Key Features}}\label{write__genotypes__knn_8h_Features}

\begin{DoxyItemize}
\item Determines whether two haplotypes are consistent with a given genotype.
\item Uses a helper function to validate allele compatibility.
\item Works with a vector-\/based representation of haplotypes and genotypes.
\end{DoxyItemize}\hypertarget{write__genotypes__knn_8h_Preconditions}{}\doxysubsection{\texorpdfstring{Preconditions}{Preconditions}}\label{write__genotypes__knn_8h_Preconditions}

\begin{DoxyItemize}
\item The input {\ttfamily h1}, {\ttfamily h2}, and {\ttfamily genotype} must be vectors of the same size.
\item Each vector should represent a sequence of alleles for a particular individual.
\item Allele values should be within the expected range (typically 0, 1, or 2).
\end{DoxyItemize}
