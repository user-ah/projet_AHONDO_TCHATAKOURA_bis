This program performs genotype imputation using a k-\/nearest neighbors (KNN) approach. It reads configuration parameters from a file, loads genotype data (which may contain missing values), imputes missing values using KNN imputation, and writes the fully imputed genotype data to an output CSV file.

The imputation process involves\+:
\begin{DoxyEnumerate}
\item Reading parameters, including the value of {\ttfamily k} (number of neighbors), from a configuration file.
\item Reading genotype data from a CSV file where missing values are represented by {\ttfamily -\/1}.
\item Computing Hamming distances to find the {\ttfamily k} nearest genotypes for imputation.
\item Imputing missing genotype values using majority voting from the {\ttfamily k} nearest neighbors.
\item Saving the imputed genotype data to an output CSV file.
