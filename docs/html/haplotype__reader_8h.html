std::vector< std::pair< std::vector< int >, std::vector< int > > > readHaplotypes ( const std::string & filename)
inline

Reads haplotypes from a CSV file and stores them in a vector.

This function reads a haplotype dataset from a CSV file where each row represents a haplotype. It processes the data and stores haplotypes as pairs of vectors of integers.

Parameters
filenameThe name of the CSV file containing haplotype data.
Exceptions
std::runtime_errorIf the file cannot be opened.
std::invalid_argumentIf the file format is invalid (e.g., uneven number of rows).
Returns
A vector of pairs of haplotypes, each represented as a vector of integers.

Usage Example

std::vector<std::pair<std::vector<int>, std::vector<int>>> haplotypes = readHaplotypes("haplotypes.csv");
for (const auto& pair : haplotypes) {
std::cout << "Haplotype 1: ";
for (int allele : pair.first) std::cout << allele << " ";
std::cout << "\nHaplotype 2: ";
for (int allele : pair.second) std::cout << allele << " ";
std::cout << std::endl;
}
