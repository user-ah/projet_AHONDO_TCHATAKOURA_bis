inline

Reorders haplotypes to match the order of genotypes and writes them to a file.

This function ensures that haplotypes are reordered to align with the corresponding genotypes and then writes them to a CSV file for further processing.

Parameters
genotypesA list of genotypes, each represented as a vector of integers.
genotypeToHaplotypesA map associating each genotype with its corresponding haplotype pair.
haplotypesFileThe name of the output CSV file where the reordered haplotypes will be written.
Exceptions
std::runtime_errorIf the output file cannot be opened for writing.
std::invalid_argumentIf the input data is improperly formatted or missing expected entries.
Returns
void

Usage Example

std::vector<std::vector<int>> genotypes = {{0,1,1}, {1,0,1}};
std::unordered_map<std::vector<int>, std::pair<std::vector<int>, std::vector<int>>, VectorHash> genotypeToHaplotypes;
genotypeToHaplotypes[genotypes[0]] = {{0,1,0}, {1,1,1}};
genotypeToHaplotypes[genotypes[1]] = {{1,0,1}, {0,1,1}};
reorganizeHaplotypesAndWriteToFile(genotypes, genotypeToHaplotypes, "ordered_haplotypes.csv");
