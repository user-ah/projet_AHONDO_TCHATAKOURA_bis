void imputeMissingData ( std::vector< int > & target,
const std::vector< std::vector< int > > & genotypes,

inline

Imputes missing values in a genotype vector using a KNN-based approach.

This function identifies the k nearest neighbors based on the Hamming distance between the target vector and all genotype vectors in the dataset. Missing values (-1) in the target vector are replaced using a majority vote from the corresponding positions in the nearest neighbors.

Parameters
targetThe genotype vector with missing values to be imputed.
genotypesA matrix of genotype vectors to use for nearest neighbor comparison.
kThe number of nearest neighbors to consider for imputation.
Exceptions
std::invalid_argumentIf target and genotype vectors have different sizes.
std::invalid_argumentIf k is not positive or exceeds the number of available genotype vectors.
Returns
void (Modifies the target vector in-place).

Usage Example

std::vector<int> target = {0, -1, 1, -1, 0};
std::vector<std::vector<int>> genotypes = {
{0, 1, 1, 0, 0},
{0, 0, 1, 1, 0},
{0, 1, 1, 1, 0}
};
int k = 2;
imputeMissingDataKNN(target, genotypes, k);
for (int val : target) {
std::cout << val << " ";
}
// Output: 0 1 1 1 0 (example result based on majority vote)
