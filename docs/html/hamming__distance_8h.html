inline

Computes the Hamming distance between two vectors, ignoring missing values (-1).

This function calculates the Hamming distance by counting the number of differing positions between two integer vectors while ignoring positions where either vector contains -1. This allows for flexibility in handling missing data.

Parameters
aThe first vector of integers.
bThe second vector of integers.
Exceptions
std::invalid_argumentIf the input vectors are not of the same size.
Returns
The computed Hamming distance between the two vectors.

Usage Example

std::vector<int> vec1 = {0, 1, -1, 1, 0};
std::vector<int> vec2 = {0, -1, 1, 1, 1};
int distance = hammingDistance(vec1, vec2);
std::cout << "Hamming Distance: " << distance << std::endl; // Output: 2
