Detailed Description

This header file defines a function to read genotype data from a text file. The genotype data is stored in a 2D vector of integers. Each row in the file corresponds to a row in the matrix, and non-numeric characters are ignored.

Authors
Ahamed TCHATAKOURA Mawu√©na AHONDO

Key Features:

  • Reads a genotype matrix from a text file.
  • Parses numeric characters and ignores other characters.
  • Supports matrices of arbitrary dimensions.
Parameters

genotypesA 2D vector to store the genotype data.

void
Precondition
The input file must be in a readable format (one genotype per cell, rows separated by newlines).
Non-numeric characters are ignored during parsing.
