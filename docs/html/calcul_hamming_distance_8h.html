Detailed Description

Computes the Hamming distance between two haplotypes.

This header file declares the function calculateHammingDistance, which computes the Hamming distance between two haplotypes represented as vectors of integers. The Hamming distance is the number of positions where the corresponding elements of the two vectors differ.

The function ensures that both input haplotypes have the same length before performing the calculation. If they have different lengths, an exception is thrown.

Key Features

  • Computes the Hamming distance between two haplotypes.
  • Validates that both input vectors have the same size.
  • Returns an integer representing the number of differing positions.

Preconditions

  • The two input haplotypes must have the same size (number of loci).
Parameters
haplo1The first haplotype, represented as a vector of integers.
haplo2The second haplotype, represented as a vector of integers.
Exceptions
std::invalid_argumentIf the two haplotypes have different sizes.
Returns
The Hamming distance (integer) between the two haplotypes.

Usage Example

std::vector<int> haplo1 = {1, 0, 1, 1};
std::vector<int> haplo2 = {1, 1, 0, 1};
int distance = calculateHammingDistance(haplo1, haplo2);
std::cout << "Hamming Distance: " << distance << std::endl; // Output: Hamming Distance: 2
