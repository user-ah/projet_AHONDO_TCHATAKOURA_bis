Detailed Description

Defines a function to read simulation parameters from a text file.

This header file contains the declaration of a function that reads the percentage of missing data (percent_miss) from a specified text file and assigns it to a variable.

Authors
Ahamed TCHATAKOURA & Mawu√©na AHONDO

The function parses a text file to find a specific parameter (percent_miss), extracts the value associated with the parameter, converts it to a float, and assigns it to the provided variable. It also handles errors if the file cannot be opened or if the parameter is missing.

Parameters
parametersFileThe path to the text file containing the simulation parameters.
percentMissA reference to a float variable where the percentage of missing data will be stored.
Precondition
The input file must include a line specifying percent_miss in the format: percent_miss=<value>.
Exceptions
std::runtime_errorif the file cannot be opened.
Note
The function exits the program with a failure status if the file cannot be opened.
